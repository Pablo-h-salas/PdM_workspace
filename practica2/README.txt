                          Se implementa un modulo de software para trabajar con retardos no bloqueantes.
                          La funcion principal se encuentra dentro de un bucle infinito, esta funcion 'LED_parpadeo' permite establecer el tiempo de encendido  
                          y apagado de un LED y la cantidad de veces que esa accion se realiza (ciclos). el tiempo de encendido/apagado depende del DUTY y el periodo indicado.
                          La funcion devuelve 'true' si se han cumplido con la cantidad de ciclos solicitada, caso contrario 'false'. 
                          Una vez que devuelve 'true' debe pasar a otro patron/modo de parpadeo, esto lo determina la variable 'mode' que incrementa cada vez que 'LED_parpadeo' devuelve true
